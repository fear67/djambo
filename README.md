**PC SHOP CONPUTER** — это сайт для *энтузиастов железа*. Проект позволяет просматривать каталог комплектующих и изучать пользовательские сборки с автоматическим расчетом стоимости, а также создавать свои сборки и выкладывать их на ***общее обозрение***.

> «Сборка компьютера — это **не просто** подбор железа, это создание ***инструмента для будущих побед***».  
> — *Команда CONPUTER.*

##  Начало работы

Инструкции ниже помогут вам развернуть проект на локальной машине для разработки или тестирования. <u>Обязательно проверьте версию Python перед началом.</u>

###  Необходимые условия

Для запуска проекта вам потребуются:
* **Python 3.10** или выше ~~(Python 3.9 не поддерживается)~~
* **Django 5.0+**
* **Библиотека Pillow** (для обработки изображений)
