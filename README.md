 **PC SHOP CONPUTER** — это сайт для *энтузиастов железа*. Проект позволяет просматривать каталог комплектующих и изучать пользовательские сборки с автоматическим расчетом стоимости, а также создавать свои сборки и выкладывать их на ***общее обозрение***. Также предоставляется возможность заказать свою собственную сборку или готовую сборку, созданную другими людьми.

 ## Начало работы 
  
 Инструкции ниже помогут вам развернуть проект на локальной машине для разработки или тестирования. <u>Обязательно проверьте версию Python перед началом.</u> 
  
 ### Необходимые условия 
  
 Для запуска проекта вам потребуются: 
 * **Python 3.10** или выше ~~(Python 3.9 не поддерживается)~~ 
 * **Django 5.0+** 
 * **Библиотека Pillow** (для обработки изображений) 
  
 ### Установка и первый запуск 
  
 Для успешного деплоя выполните следующие шаги **строго по порядку**: 
  
 1. **Клонируйте репозиторий** удобным для вас способом: 
    ```bash 
    git clone https://github.com/your-team/pc-shop-constructor.git 
    ``` 
 2. **Создайте и активируйте виртуальное окружение**: 
    - *На Windows:* 
      ```bash 
      python -m venv venv 
      .\venv\Scripts\activate 
      ``` 
    - *На macOS/Linux:* 
      ```bash 
      python3 -m venv venv 
      source venv/bin/activate 
      ``` 
 3. **Установите зависимости** (список библиотек): 
    ```bash 
    pip install django pillow 
    ``` 
 4. **Примените миграции** базы данных: 
    ```bash 
    python manage.py migrate 
    ```
